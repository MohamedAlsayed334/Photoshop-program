## 📝 Description

**Mini-Photoshop** is a console-based image processing application developed as a structured programming project in C++. The program allows users to apply various filters and effects to images, similar to basic Photoshop functionality. It uses a custom image library provided by the university to handle image loading, saving, and pixel manipulation.

### Key Highlights:
- 🖼️ Supports multiple image formats (jpg, bmp, png, tga)
- 🔄 Full undo/redo functionality
- 🎨 20+ creative filters and effects
- 📱 User-friendly menu system
- ⚡ Efficient algorithms for real-time processing

---

## ✨ Features

### 🎨 **Basic Filters**
- ✅ Grayscale Conversion
- ✅ Black & White Conversion
- ✅ Invert Colors
- ✅ Darken/Lighten Image
- ✅ Adjust Saturation

### 🔄 **Transformations**
- ✅ Flip Image (Vertical/Horizontal)
- ✅ Rotate Image (90°, 180°, 270°)
- ✅ Resize Image (Shrink/Stretch)
- ✅ Crop Image
- ✅ Add Frames (Simple & Fancy)
- ✅ Horizontal Sheer by 40 (A.K.A. Skew Filter)

### 🎭 **Creative Effects**
- ✅ Oil Painting Effect
- ✅ Edge Detection
- ✅ Blur Effect
- ✅ Infrared Effect
- ✅ Ultraviolet Effect

### 🌟 **Special Filters (Wano Series)**
- ✅ Wano Sunlight (Warm yellow tones)
- ✅ Wano Night (Purple/blue tones)
- ✅ Old TV Effect (with distortion & noise)

### 🛠️ **Advanced Features**
- ✅ Image Merging
- ✅ Frame Styling (Gradient, Checkerboard, Stars)
- ✅ Complete Undo/Redo System
- ✅ Multiple Save Options
